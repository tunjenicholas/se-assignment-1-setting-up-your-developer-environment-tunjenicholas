Operating System (OS): Windows 11

1. Create a Bootable USB Drive

Step 1: Download the Windows 11 ISO

* Navigate to Windows 11 Download.
* Under "Download Windows 11 Disk Image (ISO)," select "Windows 11" and click "Download."
* Choose your language and confirm.
* Click the "64-bit Download" button to download the ISO file.

Step 2: Download and Install Rufus

* Go to the Rufus website and download the latest version of Rufus.
* Run the Rufus executable after downloading.

Step 3: Create the Bootable USB Drive

* Insert a USB drive with at least 8GB capacity into your computer.
* Open Rufus.
* In Rufus, select your USB drive under "Device."
* Click "Select" and choose the Windows 11 ISO file you downloaded.
* Ensure the following settings are selected:
* Partition scheme: GPT
* Target system: UEFI (non CSM)
* File system: NTFS
* Click "Start" and wait for the process to complete.

2. Install Windows 11 from the Bootable USB Drive

Step 1: Boot from the USB Drive

* Insert the bootable USB drive into the PC where you want to install Windows 11.
* Restart the PC.
* During the boot process, enter the BIOS/UEFI settings (usually by pressing a key like F2, F12, DEL, or ESC).
* Set the USB drive as the primary boot device and save the changes.
* Restart the PC to boot from the USB drive.

Step 2: Start the Windows 11 Installation

* Once the PC boots from the USB drive, the Windows 11 setup will start.
* Select your language, time and currency format, and keyboard layout, then click "Next."
* Click "Install Now."

Step 3: Enter Product Key (Optional)

* If you have a product key, enter it and click "Next." If not, click "I don't have a product key" to proceed.

Step 4: Select Windows Version

* Select the version of Windows 11 you want to install and click "Next."

Step 5: Accept License Terms

* Read and accept the license terms, then click "Next."

Step 6: Choose Installation Type

* Choose "Custom: Install Windows only (advanced)" to perform a clean installation.

Step 7: Select the Drive to Install Windows

* Select the partition where you want to install Windows 11. If you have no partitions or want to start fresh, you can delete any existing partitions and create a new one.
* Click "Next."

Step 8: Complete the Installation

* The installation process will start. Your PC will restart several times during the process.
* Follow the on-screen instructions to set up your region, keyboard layout, and other preferences.
* Set up your user account and customize your settings.

Finalizing Installation

* Once the setup is complete, Windows 11 will boot to the desktop.
* Remove the USB drive.


Install a Text Editor or Integrated Development Environment (IDE): Visual Studio Code

1. Download Visual Studio Installer

Step 1: Visit the Visual Studio Website

* Navigate to the Visual Studio Download page.

Step 2: Choose the Version

*Select the version of Visual Studio you want to install. There are three main editions:
* Visual Studio Community (Free for individual developers, open-source projects, academic research, and small professional teams)
* Visual Studio Professional (Paid, for small teams)
* Visual Studio Enterprise (Paid, for large teams with advanced needs)
* Click on the "Free download" button under the desired edition (e.g., Community).

2. Run the Visual Studio Installer

Step 1: Open the Installer

* Once the download is complete, open the downloaded file (usually named vs_installer.exe).

Step 2: Start Installation

* The Visual Studio Installer will open. Click "Continue" to start the installation.

3. Select Workloads

Step 1: Choose Workloads

* The installer will present you with a list of workloads, which are collections of related tools and libraries for specific types of development (e.g., ASP.NET, Python, desktop development, etc.).
* Select the workloads that match your development needs. For example:
* ASP.NET and web development: For web development using ASP.NET.
* Python development: For Python development.
* Desktop development with C++: For C++ desktop applications.
* Data storage and processing: For database and data science applications.

Step 2: Individual Components (Optional)

* If you want more control over the specific components to install, click on the "Individual components" tab and select the items you need.

Step 3: Language Packs (Optional)

* Click on the "Language packs" tab to select additional languages if you need to develop in languages other than English.

4. Install and Launch Visual Studio

Step 1: Install

* After selecting the workloads and components, click the "Install" button.
* The installer will download and install the selected components. This may take some time depending on the number of components and your internet speed.

Step 2: Launch Visual Studio

* Once the installation is complete, you can launch Visual Studio directly from the installer by clicking the "Launch" button.

5. Initial Setup

Step 1: Sign In (Optional)

* When Visual Studio first launches, you will be prompted to sign in with a Microsoft account. Signing in is optional but recommended for syncing settings across devices.

Step 2: Choose Development Settings

* Select your preferred development settings. This configures the IDE layout and key bindings according to your chosen environment (e.g., General, C#, Web, etc.).

Step 3: Start a New Project

*You can now start a new project or open an existing one by selecting the appropriate option from the start page.


Set Up Version Control System:
1. Install Git
Step 1: Download Git

* Navigate to the Git download page.
* Click on "Windows" to download the Git installer for Windows.

Step 2: Run the Git Installer
* Open the downloaded Git installer (Git-<version>-64-bit.exe).
* Follow the installation steps, using the default settings unless you have specific preferences.

Installation Steps:
* Select Destination Location: Leave the default location or choose another folder, then click "Next."
* Select Components: Leave the default components selected, then click "Next."
* Start Menu Folder: Leave the default, then click "Next."
* Choosing the default editor used by Git: Select your preferred text editor (e.g., "Use Visual Studio Code as Git's default editor") and click "Next."
* Adjusting your PATH environment: Select "Git from the command line and also from 3rd-party software" and click "Next."
* Choosing the SSH executable: Leave the default ("Use bundled OpenSSH") and click "Next."
* Choosing HTTPS transport backend: Select "Use the OpenSSL library" and click "Next."
* Configuring the line ending conversions: Select "Checkout Windows-style, commit Unix-style line endings" and click "Next."
* Configuring the terminal emulator to use with Git Bash: Select "Use MinTTY (the default terminal of MSYS2)" and click "Next."
* Choosing the default behavior of 'git pull': Select "Default (fast-forward or merge)" and click "Next."
* Choosing a credential helper: Select "Git Credential Manager" and click "Next."
* Configuring extra options: Leave the default options checked and click "Next."
* Configuring experimental options: Leave the default options unchecked and click "Install."

Step 3: Verify Git Installation
* Open the Command Prompt or Git Bash.
* Type git --version and press Enter.
* You should see the installed version of Git.

2. Configure Git

Step 1: Set Your Username and Email
* Open the Command Prompt or Git Bash.
* Configure your username:  git config --global user.name "Your Name"
* Configure your email: git config --global user.email "your.email@example.com"

Step 2: Verify Configuration
* To verify your configuration, type: git config --list

3. Create a GitHub Account
Step 1: Sign Up for GitHub
* Go to GitHub.
* Click "Sign up" in the top right corner.
* Follow the instructions to create your account.

4. Initialize a Git Repository and Make Your First Commit
Step 1: Create a New Project Folder
* Create a new folder for your project. For example, create a folder named MyProject on your desktop.

Step 2: Initialize Git Repository
* Open the Command Prompt or Git Bash.
* Navigate to your project folder: cd path/to/your/MyProject
* Initialize the Git repository: git init

Step 3: Create a README File
* Create a README file to document your project: echo "# MyProject" > README.md

Step 4: Add Files to the Repository
* Add the README file to the repository: git add README.md

Step 5: Make Your First Commit
* Commit the file to the repository: git commit -m "Initial commit"

5. Push Your Repository to GitHub
Step 1: Create a New Repository on GitHub
* Go to your GitHub account.
* Click on the "New" button to create a new repository.
* Enter the repository name (e.g., MyProject).
* Optionally, add a description and select the visibility (public or private).
* Do not initialize the repository with a README, .gitignore, or license.
* Click "Create repository."

Step 2: Add Remote Repository
*In your Command Prompt or Git Bash, add the GitHub repository as a remote: git remote add origin https://github.com/yourusername/MyProject.git

Step 3: Push Your Local Repository to GitHub
* Push your local repository to GitHub: 
                                       git branch -M main
                                       git push -u origin main


Installing Necessary Programming Languages and Runtimes
Installing Python

Step 1: Download Python
* Navigate to the official Python website: Python Downloads
* The website should automatically detect your operating system and present the latest version of Python for Windows. Click the "Download Python x.x.x" button.

Step 2: Run the Python Installer
* Once the installer is downloaded, open the file (usually named python-<version>.exe).

Step 3: Start the Installation Process
* When the installer window opens, check the box that says "Add Python to PATH" at the bottom. This is crucial for running Python from the command line.
* Click on "Customize installation" if you want to select specific components or "Install Now" for a standard installation.

Step 4: Customize Installation (Optional)
* If you choose "Customize installation," you can select optional features such as pip (Python's package installer), documentation, and more.
* Click "Next" after selecting the desired options.

Step 5: Advanced Options
* On the next screen, you can customize advanced options such as installing Python for all users or choosing the installation directory.
* Ensure "Add Python to environment variables" is checked.
* Click "Install" to start the installation process.

Step 6: Complete the Installation
* Once the installation is complete, you will see a "Setup was successful" message.
* Click "Close" to exit the installer.

Step 7: Verify Installation
* Open Command Prompt.
* Type python --version and press Enter. You should see the installed version of Python.
* Type pip --version and press Enter to verify that pip is installed and available.


Installing Package Managers
1. Installing pip (Python)

Step 1: Verify Python Installation
* Open Command Prompt.
* Type python --version and press Enter. Ensure you have Python installed.

Step 2: Check if pip is Already Installed
* Type pip --version and press Enter. If pip is already installed, you will see the version number.

Step 3: Installing pip (if not already installed)
* If pip is not installed, you can download get-pip.py to install it.
  * Download the get-pip.py file by navigating to get-pip.py.
  * Save the file in a directory of your choice (e.g., Downloads).
* Open Command Prompt and navigate to the directory where get-pip.py is saved: cd path\to\directory
* Run the following command to install pip: python get-pip.py

Step 4: Verify pip Installation
* After the installation is complete, verify it by typing pip --version in the Command Prompt.


Configuring a Database: Installing MySQL on Windows 11
1. Download MySQL Installer
Step 1: Visit the MySQL Download Page
* Navigate to the MySQL Downloads page.

Step 2: Choose the Installer
* Click on "Download" under "Windows (x86, 32-bit), MSI Installer".

Step 3: Select the Version
* Choose the desired version (e.g., MySQL Installer 5.7.34).
* Click on "Download".

Step 4: No Thanks, Just Start My Download
* On the next page, you might be prompted to sign up for an Oracle Web account. Click "No thanks, just start my download" at the bottom.

2. Install MySQL
Step 1: Run the Installer
* Once the download is complete, open the downloaded file (usually named mysql-installer-community-<version>.msi).

Step 2: Setup Type
* The installer will start. You will see various setup types:
* Developer Default: Installs MySQL server, MySQL Workbench, MySQL Shell, and more.
* Server Only: Installs only the MySQL server.
* Client Only: Installs only the MySQL client applications.
* Full: Installs all components.
* Custom: Allows you to choose individual components.
* Choose "Developer Default" and click "Next".

Step 3: Check Requirements
* The installer will check for any missing requirements and prompt you to install them if needed. Click "Execute" to install the required software.

Step 4: Installation Progress
* The installer will show the list of products to be installed. Click "Execute" to begin the installation.
* Wait for the installation to complete, and click "Next".

Step 5: Product Configuration
* The installer will guide you through the configuration of the installed products. Click "Next".
3. Configure MySQL Server

Step 1: Type and Networking
* Choose the "Standalone MySQL Server / Classic MySQL Replication" option.
* Click "Next".

Step 2: Authentication Method
* Select "Use Strong Password Encryption for Authentication (RECOMMENDED)".
* Click "Next".

Step 3: Accounts and Roles
* Set the MySQL Root Password:
* Enter and confirm a strong password for the root user.
* Optionally, add additional MySQL users by clicking "Add User".
* Click "Next".

Step 4: Windows Service
* Configure MySQL as a Windows Service:
* Ensure "Configure MySQL Server as a Windows Service" is checked.
* Optionally, set the Windows Service Name (default is fine).
* Choose "Standard System Account" for running the service.
* Click "Next".

Step 5: Apply Configuration
* Review the configuration settings and click "Execute" to apply them.
* Wait for the configuration to complete and click "Finish".

4. Complete Installation
Step 1: MySQL Workbench and MySQL Shell
* The installer will show you the list of installed products. You can choose to start MySQL Workbench and MySQL Shell from here.
* Click "Finish" to exit the installer.

5. Verify Installation
Step 1: Open MySQL Workbench
* Open MySQL Workbench from the Start menu.

Step 2: Connect to MySQL Server
* In MySQL Workbench, click on the "Local instance MySQL" connection.
* Enter the root password you set during installation and click "OK".

Step 3: Verify Connection
* If connected successfully, you will see the MySQL Workbench interface where you can start managing your databases.


Exploring Extensions and Plugins for Visual Studio Code
Visual Studio Code (VS Code) is a popular text editor that supports a wide range of extensions and plugins to enhance its functionality. 
Below are the steps to explore and install extensions for VS Code.

Step 1: Open Visual Studio Code
* Launch VS Code
* Open Visual Studio Code from the Start menu or by searching for it in your system's application launcher.

Step 2: Access the Extensions Marketplace
* Open the Extensions View
* Click on the Extensions icon in the Activity Bar on the side of the window. It looks like a square icon with four squares inside it.
* Alternatively, you can open the Extensions view by pressing Ctrl+Shift+X on your keyboard.

Step 3: Browse and Search for Extensions
1. Browse Popular Extensions
 * The Extensions view shows a list of popular and recommended extensions. You can scroll through this list to explore extensions that might be useful for your development needs.
2. Search for Specific Extensions
 * Use the search bar at the top of the Extensions view to find specific extensions. For example, if you want to install a Python extension, type “Python” into the search bar and press Enter.
Step 4: Install Extensions
1. Select an Extension
* Click on an extension in the list to view its details. This will open a new pane with information about the extension, including its features, screenshots, and user reviews.
2. Install the Extension
* Click the “Install” button to install the extension.
* Wait for the installation to complete. You may need to reload VS Code for some extensions to take effect.

Step 5: Recommended Extensions for Various Functions
1. Syntax Highlighting and Code Formatting
* Python: Microsoft’s official Python extension. Search for “Python” and install the one by Microsoft.
* Prettier: A popular code formatter that supports many languages. Search for “Prettier - Code formatter” and install it.

2. Linting
* ESLint: A linter for JavaScript and TypeScript. Search for “ESLint” and install the extension by Dirk Baeumer.
* Pylint: A linter for Python. Often works with the Python extension by Microsoft.

3. Version Control Integration
* GitLens: An extension to enhance the built-in Git capabilities in VS Code. Search for “GitLens” and install the one by Eric Amodio.
* Git Graph: A visualizer for Git repositories. Search for “Git Graph” and install it.

4. Docker Integration
* Docker: Adds capabilities for managing Docker containers and images. Search for “Docker” and install the one by Microsoft.

5. Database Tools
* SQLTools: A database management extension for working with SQL databases. Search for “SQLTools” and install it.

6. Other Useful Extensions
* Live Server: Launches a local development server with live reload. Search for “Live Server” and install it.
* Debugger for Chrome: Adds debugging capabilities for Chrome. Search for “Debugger for Chrome” and install it.

Step 6: Configure Extensions
1. Access Extension Settings
* Click the gear icon next to the installed extension in the Extensions view and select “Extension Settings”.
* Configure the settings as per your project requirements.

2. Configure Global Settings
* Go to File > Preferences > Settings (or press Ctrl+,).
* Use the search bar to find specific settings for your installed extensions and customize them as needed.

Step 7: Manage Extensions
1. Enable or Disable Extensions
* Right-click on an installed extension in the Extensions view and choose “Disable” or “Enable” as needed.
2. Uninstall Extensions
* Right-click on an extension in the Extensions view and select “Uninstall” if you no longer need it.
